# Login Attempt Limiter

## Overview

This Python project implements a secure login system that restricts repeated incorrect password attempts. After a fixed number of failed login attempts, the account is temporarily locked before allowing further attempts.

## Features

* Username and password authentication
* Maximum failed attempt limit
* Temporary account lockout
* Automatic reset after lock period
* Simple command-line interface

## Technologies Used

* Python 3
* Time Module

## How to Run

```bash
python login_system.py
```

Enter the username and password when prompted.
